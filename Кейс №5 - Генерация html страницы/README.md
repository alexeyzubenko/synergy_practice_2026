# Кейс-задача № 5. Генерация HTML-страницы

## Описание задания
Написать программу, которая генерирует HTML-страницу с:
- Заголовком «Наименование организации на базе, которой Вы проходите практическую подготовку»
- Информацией об организации
- CSS-стилями (официальный шрифт компании + 5 альтернативных вариантов)

## Реализация
Программа написана на языке **Go**.

Основной файл генератора: `x5tech_html_generator.go`

## Запуск

cd "Кейс 5 - Генерация HTML-страницы/GO html generator"
go run x5tech_html_generator.go
После выполнения будет создан файл x5_tech_generated.html.
