## Выполненные кейс-задачи

В рамках индивидуального задания были выполнены следующие практические кейсы:

### Кейс № 4 — Юнит-тесты с GoogleTest
Реализованы юнит-тесты для классов:
- `Queue` (очередь) — методы `push`, `pop`
- `Heap` (куча) — методы `push`, `pop`
- `BinaryTree` (бинарное дерево) — методы `push`, `pop`, `search`


### Кейс № 5 — Генерация HTML-страницы
Разработана программа на **Go**, которая генерирует HTML-страницу с информацией об организации прохождения практики.  
Добавлены CSS-стили с официальным шрифтом компании + 5 альтернативных вариантов шрифтов.

## Как запустить проекты

Инструкции по сборке и запуску находятся в `README.md` внутри каждой папки кейса.
